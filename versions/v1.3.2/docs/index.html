FEATURES

九大核心功能亮点

从模式切换到规则管理，ClashOmega 让代理控制变得简单直观

🔄

三模式切换

系统代理 → 直连 → Clash 代理，一键切换，图标颜色实时变化

🔍

域名匹配检测

检测当前域名匹配的 Clash 规则，显示匹配的分组和策略

⚡

快捷添加规则

将当前域名一键添加到代理组，动态获取 Clash 代理组列表

🧠

智能域名分组

检测页面所有域名，自动分组建议（如 i1.art.com, i2.art.com → *.art.com）

📊

系统代理状态

实时显示 Windows 系统代理状态，通过 Native Host 读取注册表

📝

规则管理

查看、添加、删除 Clash YAML 配置文件中的规则，增删后自动热重载

🔥

重启 Clash

一键将 profile 规则同步到 Clash Verge Rev 快照文件并重启内核

🌍

多语言

简体中文 / English / 日本語，三语言无缝切换

🎨

Material Design 3

Google MD3 设计风格，支持亮色/暗色/玻璃拟态多主题

INSTALL

三步快速安装

加载扩展 + 安装 Native Host + 配置 API，即可使用

1

加载扩展

打开 chrome://extensions/，开启开发者模式，加载 extension/ 目录

2

安装 Native Host

右键运行 install.ps1，粘贴扩展 ID，完成注册

3

配置 API

点击扩展图标 → 设置，填写 Clash API 地址和密钥
